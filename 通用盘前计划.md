## 六、买入条件

必须同时满足：

1. 账户状态允许开仓。
2. 大盘不高开低走，成交额没有快速萎缩。
3. 主线方向清晰，板块前排主动放量。
4. 目标股回踩承接强，不是直线冲高。
5. 买入前能写清楚买入理由、仓位、止损位和失效条件。

**不交易条件**：

- 情绪急躁，想追回错过的涨幅。
- 指数冲高回落并放量。
- 板块核心股集体炸板或大幅回落。
- 买入理由只剩“别人都在涨”。
- 无法接受止损。

## 七、卖出与止损

| 场景 | 操作 |
|---|---|
| 新开仓亏损5% | 无条件止损 |
| 滚动操作跌破5日线 | 优先退出或减仓 |
| 突发大跌 | 至少减半仓 |
| 周阴线超过8% | 清仓休息 |
| 连续两周收阴 | 退出休息，重新评估 |
| 高位放大量收阴 | 清仓或大幅减仓 |

**卖出原则**：止损点到了直接执行，不纠缠。先控制亏损，再讨论收益。
