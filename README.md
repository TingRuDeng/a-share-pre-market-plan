# A 股公开盘前计划

本仓库保存可公开分享的脱敏盘前计划，并提供本地操盘面板用于查看公开计划和行情数据。

## 目录结构

- `plans/YYYY/MM/YYYY-MM-DD.md`：每日公开盘前计划。
- `index.html`、`server.js`、`assets/`：本地操盘面板。
- `docs/dev-notes.md`：开发记录和验证小结。

## 本地查看

```bash
node server.js
```

启动后打开 `http://127.0.0.1:8787`。

## 公开边界

公开计划只保留板块、方向、风险和纪律等聚合判断，不包含实盘持仓、成本、账户资产、作者身份、打赏话术、比赛账户、原文摘句和本地路径。
